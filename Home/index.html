Conceitos Básicos

Hardware

São as peças físicas do computador: teclado, mouse, monitor, placa-mãe, etc.

Software

São os programas que rodam no computador, como o navegador ou o sistema operacional.

Scripts

Pequenos códigos que automatizam tarefas, como instalar programas ou configurar algo.

Internet

Rede mundial que conecta dispositivos para trocar informações.

CPU

É o “cérebro” do computador, onde os cálculos e decisões acontecem.

GPU

Placa que cuida da parte gráfica, usada em jogos, vídeos e design.

Memória RAM

Memória temporária usada para guardar dados enquanto programas estão abertos.

HD / SSD

São os dispositivos de armazenamento onde ficam salvos os arquivos e programas do computador.

Rede Local

Conexão entre dispositivos próximos, como em casa ou no escritório.

Firewall

Protege o computador de acessos indesejados, como um segurança digital.

Antivírus

Programa que protege o computador contra vírus e malwares.

Backup

Cópia de segurança dos arquivos, para evitar perdas em caso de problemas.
